# Praveen K - Portfolio Website

A modern, premium personal portfolio website built with HTML, CSS, and JavaScript.

## Features
- **Responsive Design**: Works on all devices.
- **Dark Mode & Neon Theme**: High-end aesthetic.
- **Scroll Animations**: Smooth reveal effects.
- **Interactive Elements**: Hover effects, cursor glow.

## How to Run
Simply open `index.html` in your web browser.

## Customization
- Edit `index.html` to update content.
- Modify `style.css` to change colors or layout.
- Update `script.js` for animation tweaks.
